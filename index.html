About Tornyx

Tornyx is an experimental programming language focused on simplicity, readability, and lightweight tools.

It allows developers and learners to experiment with programming concepts while building small utilities and scripts quickly.

The project is currently in Version V4.0.1-Alpha.

Tornyx Features

Simple Syntax
Readable and beginner-friendly.
Lightweight
Minimal setup and fast execution.
Experimental
Designed for learning and innovation.
Utility Focused
Great for small tools and scripts.
